# v0.2 Migration Note

## Target
v0.1 foundation -> v0.2 stabilized foundation

## Migration Impact
1. Contract format
- `contracts/*.ts` -> `contracts/*.mjs`

2. Recipe source
- Button/Card recipe selector tanımları artık contract tabanlı generated recipe layer'da

3. Alias strategy
- Dist'te alias selector yok (plain mode)
- Geçiş için `dist/contracts/alias-migration.json` kullanılmalı

4. Theme tokens
- Light semantic token override tekrarları sadeleştirildi

## Recommended Migration Steps
1. Build al (`npm run build`)
2. `alias-migration.json` ile sınıf adlarını canonical değerlere dönüştür
3. Dist ve app usage tarafında alias class kalmadığını doğrula
4. Resolver kullanımını generated output ile karşılaştır
