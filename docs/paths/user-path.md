# User Documentation Path

> Last reviewed: 2026-04-11

Use this path if you want to consume Fikir CSS and stay inside the currently supported baseline.

## Step 1: Understand Scope
- Read `README.md` for current status and non-goals.
- Confirm support levels in `docs/roadmap/support-matrix.md`.

## Step 2: Install and Build
- Follow quick start in `README.md`.
- If using package preview flow, follow `docs/roadmap/package-distribution-strategy.md`.

## Step 3: Learn by Example
- Start from `playground/getting-started.html`.
- Review component/pattern examples in `playground/index.html`.
- Use `playground/README.md` for section map.

## Step 4: Check Quality and Limits
- Known limitations: `docs/community/known-limitations.md`.
- Release notes and migration notes: `docs/release/`, `docs/migration/`.
