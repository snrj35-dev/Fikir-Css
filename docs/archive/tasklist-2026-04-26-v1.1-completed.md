#### 23.1.2 — Dashboard starter Playwright smoke test `(P1 blocker)`

#### 23.1.3 — Form validation state contract `(P1 blocker)`
- [x] Form state contract specification doğrula

  - Input, textarea, select, checkbox state'leri `capabilities.json`'a machine-readable `states` alanı ile eklendi
  - CSS selector mapping `selectors.json` / `anatomy.json` içinde doğrulandı
  - Docs: `docs/components/input.md`, `select.md`, `textarea.md`, `checkbox.md` state examples ile güncellendi
  - Test: source contract testi + build surface testleri ile doğrulandı
  - **Dogfooding:** Test dashboard'ında form state API'si bilinmedi

#### 23.1.4 — Density impact matrix `(P1 blocker)`
