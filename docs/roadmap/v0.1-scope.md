# v0.1 Scope (Foundation)

> Status: `stale` historical scope note. Current productization baseline lives in `docs/roadmap/plan.md` and `docs/roadmap/tasklist.md`.

## In Scope
- Core + semantic tokens
- Reset + base layer
- Spacing / typography / color / layout utilities
- `stack`, `cluster`, `sidebar`, `center`, `switcher`
- `btn`, `card`, `input`, `badge`, `alert`
- Light / dark token override
- Minimal recipe API (button, card)
- Tek CSS bundle üretimi (`dist/fikir.css`)

## Out of Scope
- Tam component library
- Runtime CSS engine
- Gelişmiş compiler / static extraction
- Tam dokümantasyon sitesi

## Acceptance Criteria
- Layer sırası tek yerde tanımlı ve tüm dosyalarda tutarlı
- Utility sınıfları semantic sınıfları öngörülebilir biçimde override edebiliyor
- Build komutu `dist/fikir.css` üretiyor
- Naming config dosyası prefix modunu destekliyor

## Assumption
- v0.1 kullanıcıları manuel import/build akışını kabul eder.
