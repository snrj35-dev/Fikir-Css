# Composite Pattern Recipes

Single-file canonical examples for common UI compositions that cross multiple Fikir CSS components and primitives. Each recipe is copy-paste ready and uses only the public surface documented in `dist/contracts/anatomy.json` and `dist/contracts/primitives.json`.

## Available recipes

- `form-layout.md` — `field`/`label`/`input-group`/`helper-text`/`error-text` canonical hizalama
- `sidebar-drawer-responsive.md` — `app-shell-sidebar` ↔ `drawer` responsive geçişi (`bindSidebarDrawer` helper)
- `data-table-toolbar-pagination.md` — tablo + toolbar + sticky header + selection summary + pagination
- `skeleton-to-content.md` — `skeleton-table` → gerçek içerik geçişi ve `aria-busy` handoff
- `command-palette-binding.md` — command-palette açma, filtreleme ve keyboard nav için minimum JS binding
