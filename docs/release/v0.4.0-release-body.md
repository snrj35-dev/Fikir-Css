## Summary
v0.4.0 focuses on professionalization: support-level promotions, workflow-complete playground examples, and stronger visual/a11y quality gates.

## Highlights
- Support promotions (experimental -> supported): `textarea`, `select`, `checkbox`, `radio`, `switch`, `toast`, `pagination`
- Gap components completed: `number-input`, `rating`, `result`, `tree-view`, `tags-input`
- Workflow examples added:
  - `playground/settings-workflow-example.html`
  - `playground/tree-table-workflow-example.html`
  - `playground/result-status-example.html`
- Visual regression progression:
  - deterministic capture command: `npm run capture:playground`
  - supported-slice baseline set: `playground/screenshots/sections/*.png`
