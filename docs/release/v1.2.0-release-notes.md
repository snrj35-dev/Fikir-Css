# v1.2.0 Release Notes

This release marks the completion of the v1.2.0 "Premium & Ecosystem" milestone. It introduces high-end visual effects, an OLED theme, micro-interactions, and a limited utility layer for improved developer flexibility.

## Key Highlights

### 1. Premium Visual Layer
- **Glassmorphism:** Added `data-variant="glass"` support for `app-shell`, `modal`, and `navbar`.
- **OLED Theme:** A deep black (`#000000`) theme for high-contrast environments.
- **Micro-interactions:** Spring-based hover and active animations for buttons and links.
- **Mesh & Conic Borders:** Next-gen visual variants for `kpi-card`.

### 2. Ecosystem & Onboarding
- **Limited Utility Layer:** Spacing and layout utilities (`mt-2`, `p-4`, etc.) added while preserving the contract-driven core.
- **JS Helpers:** Released `fikir-helpers.js` for standalone interactivity (Modals, Drawers).
- **Onboarding Guides:** New "Zero to Dashboard" and "Headless UI Adapter" guides.

### 3. Architecture Hardening
- **App Shell Fullscreen:** Support for edge-to-edge dashboard layouts via `data-variant="fullscreen"`.
- **Border Soup Resolution:** Intelligent nesting rules to prevent redundant borders in complex layouts.
- **State Standardization:** Unified `aria-current="page"` and `data-active="true"` across navigation components.

## Support Status
- Typography components (`heading`, `text`, `code`, `kbd`, `quote`, `callout`) were expanded and documented in the machine-readable contracts, but remain **experimental** in the support matrix until full component docs and promotion evidence are complete.
