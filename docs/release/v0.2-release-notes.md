# v0.2 Release Notes (Foundation Stabilization)

## Summary
v0.2, yeni feature eklemeden foundation mimarisini contract-driven hale getirir.

## Highlights
- Naming source-of-truth: `contracts/naming.contract.mjs`
- Recipe source-of-truth: `contracts/recipes.contract.mjs`
- Recipe CSS + resolver generation aynı contract'tan üretilir
- Build validation pipeline genişletildi
- Alias migration artifact eklendi: `dist/contracts/alias-migration.json`
- Light semantic token tekrarları sadeleştirildi

## Behavioral Notes
- Plain modda dist sadece canonical selector class'larını içerir.
- Recipe class surface artık layer + resolver arasında tek sözleşme ile yönetilir.
