# v0.2 Release Notes (Foundation Stabilization)

## Summary
v0.2 foundation üzerine yapılan v0.2.x turlarında contract-driven mimari korunarak component surface plan fazlarına uygun biçimde tamamlandı.

## Highlights
- Naming source-of-truth: `contracts/naming.contract.mjs`
- Recipe source-of-truth: `contracts/recipes.contract.mjs`
- Recipe CSS + resolver generation aynı contract'tan üretilir
- Build validation pipeline genişletildi
- Alias migration artifact eklendi: `dist/contracts/alias-migration.json`
- Light semantic token tekrarları sadeleştirildi
- Input validation surface eklendi (`aria-invalid`, `readonly`, `disabled` ayrımı)
- Button `:disabled` davranışı ve Card attribute-slot composition yüzeyi eklendi
- Field composition surface eklendi (`field`, `label`, `helper-text`, `error-text`)
- Form controls eklendi: `textarea`, `select`, `checkbox`, `radio`, `switch`
- Input Group surface eklendi (`input-group`, `input-group-addon`)
- Overlay surface'e tooltip eklendi (`tooltip`, `tooltip-content`, `data-open`)
- Overlay surface'e popover eklendi (`popover`, `popover-content`, `data-open`)
- Overlay surface'e dropdown menu eklendi (`dropdown-menu`, `dropdown-menu-content`, `dropdown-menu-item`, `data-open`)
- Overlay surface'e drawer eklendi (`drawer`, `drawer-backdrop`, `drawer-panel`, `data-open`)
- Navigation surface başlangıcı eklendi (`tabs`, `accordion`, `pagination`)
- Navigation surface genişletildi (`breadcrumb`, `navbar`)
- Navigation surface genişletildi (`sidebar-nav`, `stepper`)
- Shell surface başlangıcı eklendi (`page-header`, `section-block`, `app-shell`)
- Data display surface başlangıcı eklendi (`table`, `empty-state`)
- Data display surface genişletildi (`avatar`, `stat`)
- Data display surface genişletildi (`timeline`, `kpi-card`)
- Structured data surface başlangıcı eklendi (`list`, `description-list`)
- Core foundation completion yüzeyi eklendi (`icon-button`, `link`, `divider`, `surface`, `visually-hidden`, `skeleton`, `spinner`) ve badge/alert refinements tamamlandı
- Form extensions yüzeyi eklendi (`range-slider`, `otp-input`)
- Overlay extensions yüzeyi eklendi (`context-menu`, `progress`, `loading-overlay`)
- Navigation & shell extensions yüzeyi eklendi (`menu-bar`, `split-pane`) ve layout primitives refinement tamamlandı (`container`, `grid`, `stack/cluster/sidebar/switcher/center` iyileştirmeleri)
- Data-grid implementation eklendi (`data-grid`, `data-grid-head`, `data-grid-body`, `data-grid-row`, `data-grid-cell`)
- Data/display extensions eklendi (`avatar-group`, `tag-chip`)
- Content / rich UI foundation yüzeyi eklendi (`text`, `heading`, `code`, `code-block`, `callout`, `quote`, `kbd`, `markdown-surface`)
- Dark mode kontrast uyumu iyileştirildi (özellikle `badge`, `alert`, `table`, `data-grid` ve white-mix kullanılan yüzeylerde)
- Search/productivity surface başlangıcı eklendi (`combobox`, `search-box`)
- Search/productivity surface genişletildi (`autocomplete`, `command-palette`)
- Product pattern surface genişletildi (`command-bar` spec+implementation, `filter-bar` implementation)
- Product pattern surface genişletildi (`data-table-toolbar` spec+implementation)
- Advanced surface'e date picker eklendi (`date-picker` component RFC+implementation)
- Advanced surface'e date range picker eklendi (`date-range-picker` component RFC+implementation)
- Advanced surface'e calendar eklendi (`calendar` component RFC+implementation)
- Advanced surface'e file upload eklendi (`file-upload` component RFC+implementation)
- Advanced surface'e dropzone eklendi (`dropzone` component RFC+implementation)
- Advanced surface'e editable field eklendi (`editable-field` component RFC+implementation)
- Overlay/advanced surface'e hover card eklendi (`hover-card` component RFC+implementation)

## Behavioral Notes
- Plain modda dist sadece canonical selector class'larını içerir.
- Recipe class surface artık layer + resolver arasında tek sözleşme ile yönetilir.
- Semantic token tabanlı dark/light uyumu güçlendirildi; white-temelli karışımlar azaltıldı.
