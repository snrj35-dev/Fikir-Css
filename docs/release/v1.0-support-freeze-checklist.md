# v1.0 Support-Level Freeze Checklist

> Created: 2026-04-12
> Scope: M4 — gates before declaring v1.0 stable

## Purpose

Before publishing `v1.0.0`, all surfaces that are `supported` must pass this freeze checklist. No surface may be added to `supported` after freeze without a patch release.

## Per-Surface Checklist

For every surface in the `supported` column of `docs/roadmap/support-matrix.md`:

- [ ] RFC/spec exists and is current (not `stale`)
- [ ] CSS implementation exists in `packages/components/*.css`
- [ ] Contract keys exist in `contracts/naming.contract.mjs`
- [ ] Entry in `scripts/css-manifest.mjs`
- [ ] `tests/build/*.surface.test.mjs` exists and passes
- [ ] Accessibility expectations are documented in the RFC
- [ ] Playground usage example exists in `playground/index.html`
- [ ] Release impact review recorded in `docs/release/promotion-evidence/`
- [ ] Support level declared in `docs/roadmap/support-matrix.md`

## Bundle Freeze Gates

- [ ] Full bundle gzip ≤ 25KB
- [ ] Core bundle gzip ≤ 14KB (after opt-in split)
- [ ] `npm run validate:size` passes
- [ ] `npm run report:contrast` passes (no AA failures)
- [ ] `npm run validate:recipe-strict` passes

## Contract Freeze Gates

- [ ] `contracts/naming.contract.mjs` has no pending deprecation warnings
- [ ] `contracts/recipes.contract.mjs` typing is complete
- [ ] `dist/contracts/alias-migration.json` covers all v0.x → v1.0 renames
- [ ] `docs/contracts/contract-compatibility-matrix.md` is current

## Documentation Gates

- [ ] README is up to date and passes `docs:validate`
- [ ] All `docs/rfcs/**` are not in `Draft` status for supported surfaces
- [ ] Migration guides exist for Tailwind, Bootstrap, MUI
- [ ] Adapter guides exist for React, Vue, Svelte

## Governance Gates

- [ ] v1.0 migration guarantee policy is published
- [ ] API stability badges are applied to all supported surfaces in support matrix
- [ ] Versioned compatibility matrix for adapters is published
- [ ] External feedback log is reviewed and all action items are resolved or deferred
