# Fikir CSS v1.0.0 Release Notes

> Released: 2026-04-20  
> Tag: `v1.0.0`  
> npm: `npm install fikir-css@1.0.0`

---

## Summary

First stable release of Fikir CSS. The `supported` surface list is now **frozen** under semver: breaking changes to any supported surface require a MAJOR version bump.

**99 total surfaces:** 69 supported · 22 beta · 12 experimental · 0 deprecated  
**Bundle:** ~152 KB raw / ~18 KB gzip

---

## What's New (M14–M21)

### Compound Patterns (CC.1)
- 12 new `data-pattern` compound components: `coachmark`, `auth-screen`, `filter-bar`, `data-table-toolbar`, `empty-search-state`, `stat-group`, `time-range-picker`, `command-bar`, `tree-table`, `onboarding-checklist`, `search-result-item`, `password-input`
- Pattern components use `[data-pattern="..."]` + `[data-slot="..."]` — no CSS class required

### Machine-Readable Contracts (M20)
- `dist/contracts/anatomy.json` — 72 component HTML anatomy entries
- `dist/contracts/capabilities.json` — per-component capability matrix
- `dist/contracts/variants.json` — 6 tones, 5 styles, 4 sizes, 19 recipe components
- `dist/contracts/primitives.json` — 7 layout primitive defaults
- `dist/contracts/selectors.json` — 429 selectors + global/per-component `data-*` markers + deprecation support
- `scripts/detect-stale-anatomy.mjs` — CI warning for undocumented components
- `npm run validate:recipe-strict` — strict recipe contract validation

### Framework Examples (M18)
- `examples/react-vite` — full runnable React + Vite starter
- `examples/vue-vite` — full Vue 3 starter with overlay demo
- `examples/svelte-vite` — full SvelteKit starter

### Quality Gates (M19, M21)
- 482 automated tests (147 source + 335 build), 0 failures
- Playwright browser test suite (236+ tests)
- `detect:stale-anatomy`, `validate:recipe-strict`, `detect:anatomy-drift` added to CI
- `validate:prefixed` fixed for pattern domain
- Bundle size thresholds updated to v1.0 baseline

### Playground (CC.2)
- Density + reduced-motion toggle buttons in playground header
- Support level badges (`supported` / `beta` / `experimental`) visible in sidebar

### Docs & Governance (M21.1–M21.5)
- `docs/release/what-is-stable-in-v1.md` — full beta/experimental surface list
- `docs/migration/v0.6-to-v1.0-migration-note.md`
- `docs/guides/troubleshooting-faq.md`
- `docs/release/hotfix-checklist.md`

## Migration from v0.6.0

No selector renames in the `supported` surface tier. See [`docs/migration/v0.6-to-v1.0-migration-note.md`](../migration/v0.6-to-v1.0-migration-note.md) for full details.

```bash
# Update your dependency
npm install fikir-css@1.0.0
```

None for `supported` surfaces. `beta` and `experimental` surfaces carry no stability guarantee.
