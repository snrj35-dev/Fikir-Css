# v0.3 Release Notes (Foundation + Surface Expansion)

## Summary
v0.3, v0.2 foundation stabilizasyonunun üzerine daha geniş component/pattern yüzeyini ve dokümantasyon-test hizasını tamamlayan sürümdür.

## Highlights
- Contract-driven mimari korunarak selector ve recipe yüzeyi genişletildi.
- Playground kapsamı foundation odaktan ürün yüzeyine yakın bir showcase seviyesine çıkarıldı.
- Data/display katmanında `table` + minimal `data-grid` birlikte sunuldu.
- Advanced surface bileşenleri eklendi: `date-picker`, `date-range-picker`, `calendar`, `file-upload`, `dropzone`, `editable-field`, `hover-card`.
- Product pattern örnekleri genişletildi: `command-bar`, `filter-bar`, `data-table-toolbar`.
- Dark mode kontrastı özellikle danger tone ve data/display yüzeylerinde iyileştirildi.
- `table-row-selected` gibi canonical olmayan class kalıntıları temizlenerek `data-row-selected="true"` standardı netleştirildi.

## Behavioral Notes
- Canonical selector yüzeyi `contracts/naming.contract.mjs` ile yönetilmeye devam eder.
- Recipe surface ve resolver çıktıları contract ile tek sözleşmede kalır.
- Demo/pattern örnekleri framework class üretmeden mevcut canonical surface üzerinden kompoze edilir.

## Upgrade Notes (v0.2 -> v0.3)
- `package.json` version: `0.2.0` -> `0.3.0`
- Dokümantasyon ve GitHub şablonları v0.3 scope diline güncellendi.
- Table selected-row örneklerinde `class="table-row-selected"` yerine `class="table-row" data-row-selected="true"` kullanılmalıdır.

## GitHub Release Checklist
1. Yerel doğrulama: `npm run test:ci`
2. Değişiklik özeti ve migration notlarını kontrol et (`docs/release/v0.3-release-notes.md`)

4. GitHub release başlığı: `v0.3.0`
5. Release body olarak bu dosyanın `Summary`, `Highlights`, `Upgrade Notes` bölümlerini kullan
