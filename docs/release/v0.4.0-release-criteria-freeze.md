# v0.4.0 Release Criteria Freeze (Draft)

> Last reviewed: 2026-04-11
> Status: frozen scope draft (publish pending)

## Included in v0.4.0 Scope
1. Support-level promotions
- `textarea`
- `select`
- `checkbox`
- `radio`
- `switch`
- `toast`
- `pagination`

2. Competitive gap components
- `number-input`
- `rating`
- `result`
- `tree-view`
- `tags-input`

3. Workflow completeness examples
- multi-step settings workflow with validation states
- tree + table mixed workflow
- result/status outcomes workflow

4. Quality automation updates
- deterministic screenshot capture command (`npm run capture:playground`)
- supported-slice visual baseline set under `playground/screenshots/sections/`

## Deferred from v0.4.0
- External feedback volume goals (`>=5`) until real submissions are collected.
- GitHub milestone/issue bootstrap actions that require valid authenticated token/session.
- Full pixel-diff visual regression in CI.
