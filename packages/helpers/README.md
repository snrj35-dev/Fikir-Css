# fikir-css/helpers — Overlay JS Utilities (M7, planned)

This package will provide vanilla JS helpers for overlay keyboard behavior and focus management.

Planned exports:
- `createFocusTrap(element)` — focus trap + return focus on close
- `bindOverlayKeyboard(element)` — Escape + backdrop click dismiss
- `createRovingTabindex(container)` — arrow key navigation
